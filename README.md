# Code_Nexus IDE 🚀

A professional-grade, multi-language code execution environment featuring secure Docker sandboxing, persistent file storage, and an integrated AI Debugger.

## ✨ Core Features
* **Multi-Language Support:** Execute JavaScript, Python, C, C++, Java, Go, Rust, PHP, and Ruby.
* **Secure Sandbox:** User code is isolated inside temporary Docker containers with strict CPU, memory, and network constraints.
* **AI Debug Assistant:** Integrated LLM analyzes compiler errors and standard output to explain and fix code.
* **Persistent Workspaces:** Full file-tree preservation via MongoDB.
* **React Frontend:** Monaco Editor integration with a responsive split-pane UI.
